CRUDlex\CRUDEntity¶

class CRUDEntity¶

Represents a single set of data in field value pairs like the row in a database. Depends of course on the {@see CRUDData} implementation being used. With this objects, the data is passed arround and validated.

The {@see CRUDEntityDefinition} defining how this entity looks like.
